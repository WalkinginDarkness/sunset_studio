# 落日间 xpaidia

#### **「落日间」是一个试图回答「何为游戏」与「游戏何为」的跨界媒体实验室**

我们正创作试图帮助我们理解游戏和人文的文字，播客，视频，游戏，寻求电子游戏联结与更多的可能。
