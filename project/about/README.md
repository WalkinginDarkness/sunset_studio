# 指北 Gide

所谓指北系列只是我们所瞥见的一系列理解世界和生活的可能，

这并不是成功学的「指南」，而是面朝未知的出走，故名“指北”

我记下行程上的路标与要前进的方向，但无法保证其结果，或许走向的是失败，但谁知道呢？

指北系列或取材于落日一周间Newsletter，区别在于写作风格、会持续更新和不定期维护。



Guide 去掉而转向 Gide，只因想起 安德烈·纪德在《人间地粮》中写

**「把书都烧掉，然后出走吧」**

\*\*\*\*

希望这些能够帮助你更好的前进
